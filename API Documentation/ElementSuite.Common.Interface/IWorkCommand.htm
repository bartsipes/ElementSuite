Type IWorkCommand

Namespace ElementSuite.Common.Interface

Defines the operation that will allow distributed work to be executed remotely. Classes that implement this interface must also place the ExportAttribute on the class and provide the type of IWorkCommand as a parameter. [Export(typeof(IWorkCommand))] public class ImplementingWorkCommand : IWorkCommand { ... }
